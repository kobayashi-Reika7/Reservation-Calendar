# ログイン手順・デモデータの作成

## 1. 事前準備

- バックエンド: `cd backend && uvicorn main:app --reload --port 8001`
- フロントエンド: `cd frontend && npm run dev` → http://localhost:5200
- Firebase Console で **Authentication → サインイン方法 → メール/パスワード** を有効にしておく

---

## 2. ログイン手順（既存ユーザー）

1. ブラウザで http://localhost:5200 を開く
2. **ログイン** 画面で次を入力する  
   - メールアドレス: 登録済みのメール  
   - パスワード: 登録済みのパスワード  
3. **「ログイン」** をクリックする  
4. 成功すると **予約日を選んでください**（カレンダー画面）に遷移する  
5. 未登録のメールの場合は「メールアドレスかパスワードが違います。」と表示される → 新規登録へ

---

## 3. 新規登録手順（初回ユーザー）

1. ブラウザで http://localhost:5200 を開く  
2. **「新規登録はこちら」** をクリックする  
3. **新規登録** 画面で次を入力する  
   - メールアドレス: 使用するメールアドレス（例: `demo@example.com`）  
   - パスワード: 6文字以上（例: `demo1234`）  
4. **「登録する」** をクリックする  
5. 成功すると **予約日を選んでください**（カレンダー画面）に遷移する  

---

## 4. デモデータの作成

1. カレンダーで **今日以降の日付** をタップする  
2. 予約入力フォームで  
   - 大分類 → 診療科 → 予約目的 → 担当医 → 時間  
   を順に選び **「内容を確認する」** をクリック  
3. 確認画面で **「予約を確定する」** をクリック  
4. 別の日付・時間で同様に 2〜3 件繰り返す  

---

## 5. 動作確認の目安

- ログイン後、ページをリロードしてもカレンダー画面のまま（認証状態が保持されている）  
- 予約確定後、該当日の予約済み枠はフォームで選択できない  
- バックエンド起動時は、ログイン成功後に `GET /users/me` でユーザー情報が同期される
