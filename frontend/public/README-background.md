# 背景画像の設定

アプリ全体の背景に画像を使うには、このフォルダ（`public/`）に **`background.png`** を置いてください。

- ファイル名: **background.png**（`src/App.css` で `url('/background.png')` を参照しています）
- 推奨: 横 800px 以上、縦 1200px 以上（スマホ縦長でカバーしやすい）
- ChatGPT などで取得した画像は、ブラウザで開いて「名前を付けて保存」で `background.png` としてこのフォルダに保存

保存後、フロントを再読み込みすると背景に表示されます。
